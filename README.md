# Dashboard Analítico - Predicción de Enfermedades

## Descripción

Proyecto desarrollado para aplicar técnicas de Data Engineering y Data Science utilizando un dataset personalizado de 3595 registros.

## Tecnologías

- Python
- Pandas
- Streamlit
- Scikit-Learn
- Matplotlib
- Seaborn

## Dashboard

El dashboard contiene:

- KPI de porcentaje de pacientes con enfermedad.
- Gráfico de barras.
- Histograma.
- Heatmap de correlación.
- Storytelling con hallazgos y recomendaciones.
